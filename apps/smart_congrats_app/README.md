# smart_congrats_app (Workspace)

This directory is the workspace root for the SmartCongrats Flutter/Dart app.

Workspace packages:
- `packages/application`
- `packages/infrastructure`
- `packages/presentation` (Flutter app)
- `packages/foundation`

Common commands:
- `flutter pub get` at the workspace root
- `dart pub get` inside a pure-Dart package (when working package-by-package)

Running the app:
- `cd packages/presentation && flutter run`
